# 一、项目介绍

CJ Labs 是我的长期创业项目。

它不仅仅是一个网站，而是未来个人品牌、AI 产品、AI 创业、AI 服务和商业生态的起点。

未来所有内容、产品、课程、工具、社区和 SaaS 都将围绕 CJ Labs 展开。

本仓库是 CJ Labs 官网的源码仓库。

# 七、设计原则

网站设计应遵循以下原则：

* 极简
* 高级
* 科技感
* 国际化
* 响应式
* 默认支持 Dark Mode
* 页面加载速度快
* SEO 友好
* GEO（生成式引擎优化）友好

整体风格参考：

* Apple
* Stripe
* Linear
* Vercel
* Notion

避免：

* 花哨动画
* 信息过载
* 国产模板风格
* 复杂配色

# 九、Cursor 开发规范

AI 在开发本项目时，应始终遵循以下原则：

## 代码

* 保持代码整洁
* 优先组件化
* 避免重复代码
* 保持良好的目录结构

## UI

* 优先现代设计
* 优先用户体验
* 页面简洁
* 留白充足
* 默认移动端适配

## 性能

* 页面尽量静态化
* 优先使用 Server Components
* 图片优化
* SEO 优先

## 开发

如果没有特殊要求：

优先采用业内最佳实践。

不要为了炫技增加复杂度。
